## Video to Audo Converter

A Python script to conver video to a mp3 file. Works on most common video formats.

To run the script, go to the directory in bash,

1. Install dependencies
```bash
pip3 install -r requirements.txt
```

2. Run the script
```bash
python3 video-to-audio.py
```

3. Enter the input video directory and output directory when prompted.
